# Food Recommendation Dataset (100,000 rows)

This file was generated for a food recommendation / multi-agent-system project.

## Important note
This is a **synthetic-but-realistic** dataset created in this environment because bulk downloading and fully reconciling multiple external food corpora was not possible here. The schema and nutrient logic were designed to align with public food/recipe data sources such as:
- DataHive `recipes-with-nutrition`
- USDA FoodData Central
- OpenNutrition / Open Food Facts
- Google Nutrition5k
- TheMealDB

## Files
- `food_recommendation_dataset_100k.csv` — main dataset
- `food_recommendation_data_dictionary.csv` — schema reference

## Best use
- meal recommendation engines
- filtering by diet, cuisine, allergens, macros, meal type
- ranking logic for multi-agent systems
- testing and evaluation pipelines

## Caveat
Use this dataset for coursework / prototyping. It should not be used for medical or clinical nutrition advice.
